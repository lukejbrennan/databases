Relational Schema

Ingredients(Name [key], Calories, Price, Amount, Recipe_Name [key], Cook Time, Directions [key], Recipe_Calories, Recipe_Price)

Tools

We will be using the main tools made available to us. We will use MySQL for the database and a combination of HTML/PHP, Javascript, and CSS for the user interface. These are obvious choices due to their relative simplicity and the lack of a need for other tools. We will use a Python scraping script to collect our data.

How We Got the Data

We are taking the data (recipes) from the website AllRecipes. We will use a scraper in the form of a Python script to do this, storing all of the collected data on dsg1.
