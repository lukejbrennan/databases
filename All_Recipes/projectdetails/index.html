Nothing In the Fridge™

About NITF

  You are hungry. You are STARVING! But you only have spaghetti sauce and noodles in your fridge. What can you make?! Who knows?! Our website does! You can input a list of ingredients and the website will return a list of recipes that only use those ingredients or it can return a list of recipes that include those ingredients. That way, even if you have Nothing in the Fridge â„¢ you can make dinner!

  Description:
  NITF™ will take an input of ingredients and with those ingredients will query a database containing a list of recipes. Each recipe will contain a list of ingredients, a name, a set of directions, and a cook time. After finding a recipe either only containing the listed ingredients or containing the listed ingredients as well as additional ingredients, the database will return the names of the recipes. The user can then click on a recipe and receive more details on the recipe.

  Why is this useful?
  We've all been in a situation in which there's food in the fridge, but we have no idea what to make for dinner. NITF™ will take your available ingredients and find a recipe to fit your needs! While there are other websites out there that perform similar actions, they fail to limit their results to only inputted ingredients.

  Where is the data coming from?
  Using a script, we will pull data from the website "AllRecipes". There is a possibility that we may decide to use a different website.

  Basic Functions
  Insert - Insert will allow for new recipes to be added into the recipe database
  Update - Update will allow for recipes to be changed. For example, if a recipe needed to be changed from needing 2 cups of salt to 2 tsp of salt.
  Delete - Delete will allow for recipes to be deleted. Say your recipe caused a minor case of food poisoning at your family Thanksgiving dinner. Delete will allow for that recipe to be deleted for all of time.

  Advanced Functions
  Note: We will probably only choose 2-3 of these ideas.

  Price Determination - This function will take an estimated guess to the cost of your meal. This will allow users to feel pretty good about their choice to eat in and make a meal.

  Calories Counter - This function will estimate the amount of calories in your meal. This will allow users who are trying to #weightloss or #carboload to keep track of their calorie count.

  Inspirational Image - This function will find photos of your website on the interwebs. If a user searched for a burger recipe, this function will search and display images for burgers. This will give our users inspiration to go out and cook their meal.

  Recipe Recommender - This function will require users to create an account. As users start to find and make recipes with our database, this function will start to develop a "knowledge" of what a particular user likes to eat. Once the user has made enough recipes, recipe recommender will start to recommend ideas for the suer before she searches for a partiular recipe.
